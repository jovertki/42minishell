# minishell

Кастомный shell на Си, bash на минималках

## Сборка и запуск
```
make
./minishell
```
## Поддерживаемые фичи

* Built-in echo, cd, pwd, export, unset, env, exit
* Запуск бинарников из $PATH
* Взаимодействие подпроцессов через pipe
* Переменные окружения
* Обработка сигналов
* Перенаправления <, >, <<
* Heredoc через >>
* Код ошибки через $?
* Обработка кавычек
* История
